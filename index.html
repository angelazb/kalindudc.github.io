Eventalicious
UX / UI Design Project

This web-based application that will enable users to connect with others in real time by projecting various nearby events and item promotions surrounding the user. Simply put, this application would inform the user of all accessible social events, special promotions or educational gatherings that are happening or will occur in the immediate future based on the geographical location of an individual.
